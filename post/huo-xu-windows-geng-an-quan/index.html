或许 Windows 更安全

Posted on 2017-06-19，4 min read

time：2017-06-19

author：wolf3c

字数：1.0K

前些日子 Windows 爆出了覆盖全球的勒索病毒 Wannacry，很多人开始叫嚣「Windows 垃圾」、「Windows 漏洞太多」、「Mac 才是最安全的电脑」、「赶紧转 Mac」……

这让我想起一句话「他们一发声，上帝就发笑」。先不说 Windows 到底是不是有那么烂，就说 Mac 是不是有那么安全呢？我估计说 Mac 很安全的人都是一些不明就里的小白，甚至只是向往 Mac 却未曾使用过的 Windows 用户。他们大概不知道早在 16 年初，大批 Mac 用户的电脑内容遭到勒索软件加密，必须是交付赎金来拿回自己的数据。

那么，Windows 和 Mac，到底哪个更安全呢？我认为是 Windows。

这可能和大多数人的印象不一致，更多时候我们听到的报道都是 Windows 又怎么了，而对于 Mac 只有褒奖之词。但有几个现实常常被忽视：

  • Windows 用户量和覆盖范围远大于其他系统：
    • 病毒研发者不论出于什么目的，研发 Windows 病毒更容易达到自己的目的。向每个 Windows 用户收 1 元，远多于向每个 Mac 用户收 10 元，同样想要攻击政府组织等，使用 Windows 绝对更容易达到目的，要炫技，让用户更多的 Windows 用户知道是不二选择。
    • Windows 网络效应更强，病毒的传播需要网络，如果你是 Mac 病毒，那意味着传播速度会变慢很多。
  • Windows 应用场景更多、app 更多。要知道大多数 ATM、医疗系统、工业设备管理系统等都是跑在 Windows 系统上，这就造就了更多可挖掘的利益点，自然吸引来更多的探险者。
  • Windows 资料更多，老师更多，更易学。要知道，所有的黑客，都是从小白开始的，如果他们选择一个没有多少资料的方向，那必然很难快速成长。

说直白点，Mac 不是更安全，只是大多数时候不值得被黑罢了。

我说 Windows 更安全，是有实际依据的。如果你看看 Windows 和 Mac 的更新频次和更新 log，随便一比较，Windows 系统方面下的功夫有目共睹，而 Mac 大概把大部分精力都放到了 iOS 上，Mac 已经好几年没有长进了。

（可以读我翻译的一篇国外友人的文章了解更多：《[译]我为什么弃 Mac 投 Windows：放弃 Apple》）

微软对于 Windows 的重视及对危机的反应速度从这次 Wannacry 病毒事件中可见一斑。

用户也是一个重要原因。

  • 对于 Windows 用户，每个人都知道一两个安全工具（不论是 360 还是 QQ 安全管家），大多数电脑上都有安全工具，如果电脑出了问题，每个人都有一套自己的方法尝试解决。估计 Mac 如果发现安全问题，用户只能等到 Apple 在百忙之中予以关注。
  • 对于 Windows 用户，打补丁和更新杀毒软件是一个常态，反观 Mac 用户，很多人都关闭了更新或从不更新。
  • 对于 Windows 用户，数据备份意识更强，这也许有点搞笑，但事实就是 Windows 用户因为经常发生重装系统等问题，很多人重要的数据是和系统分离的。

最后，由于 Mac 用户的增多，必然有一帮人会关注 Mac OS，可以预见，未来 Mac 的病毒会越来越多，而 Mac 的防护机制仅仅是 Apple 自己封闭在自己的内部，缺少更多专业安全软件公司助力，有点命悬一线的味道。

坐在狼堆中间奋力防御的 Windows 看着无狼问津的 Mac 说，「Mac 更安全？呵呵」。
