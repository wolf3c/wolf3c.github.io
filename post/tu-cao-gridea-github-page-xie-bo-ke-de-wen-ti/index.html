吐槽 Gridea + Github Page 写博客的问题

Posted on 2021-11-02，1 min read

有一段时间我没有写博客，最近几个月写了几篇，本来想发布到 GitHub page 中，发现使用 gridea 每次都同步失败。

我尝试排查了版本、token 等问题，一直无果。今天下定决心排查一下这个问题，看到有人在 issue 中提到了代理问题，尝试了一下，发现还真是这个问题。

我本想破口大骂 gridea，突然想到可能不怪他，毕竟服务端是 GitHub。

我想骂 GitHub，又发现人家不过是做了个平台，免费给我用，用户体验和挺好的，我还有什么不满足的呢。
那我应该骂什么，总不能自己命不好还怪政府吧？

想想还是骂自己吧，为什么不直接使用那些可以很容易连接并使用的平台呢，人家还会拜读我的内容，并删除我的错误，让我在正确的共同富裕道路上前进。

shit，这篇文章又出现同步错误。

fuck, error again.
