Toby：解救 Chrome 重度用户

Posted on 2017-01-03，4 min read

time：2017-01-03

author：wolf3c

字数：1000

每一个 Chrome 重度用户都要面对两座大山：

  1. 内存永远不够；
  2. 标签页不断增加。

说这是煎熬一点都不为过，系统老是提醒「内存不足」，电脑经常卡顿，标签页的按钮越来越小，切换标签越来越困难，一不小心还可能关掉正在使用的页面。

可能大家脑海中已经浮现出了几个救星的名字——「OneTab」、「The Great Suspender」、「TooManyTabs」，我自己也是尝试用这些工具「自我救赎」，然而效果差强人意：

  • OneTab 变成了一个垃圾箱，很多页面放在 OneTab 后，OneTab 也被挤在万千标签之中，后来我自己也忘记了 OneTab。
  • The Great Suspender 很不错，它可以自动 suspend 标签，可以释放部分电脑资源，但它无法解决越来越小的标签按钮带来的痛苦。
  • TooManyTabs 操作窗口太小，使用起来实在别扭。

经过一番挣扎后，我一度转回了 Firefox，Firefox 不会让标签按钮变得很小，「Tab Groups」扩展可以让我很方便地管理上百个标签页，并且 Firefox 对于电脑资源地消耗也是我可以接受的。

但是，我无法割舍 Chrome 五花八门的扩展程序，所以我常在 Chrome 网上应用店中找标签页管理工具，以期重回 Chrome 阵营。

直到我发现了「Toby」。

Toby 是一款组织、管理 Chrome 浏览器标签页的扩展程序。我使用数月后已经无法离开它，所以特地推荐给大家。

更有用的「新标签页」

Toby 会更改「新标签页」，让所有的标签页管理工作在「新标签页」中完成。

每次打开「新标签页」，都可以看到 Toby 和所有的标签页。

自由、清晰的分类

Toby 中，我们可以按照自己的喜好去建立 list，然后把标签页放在不同 list 中，达到分门别类管理标签页的目的。我自己建了「前端工具」、「PM」、「Lisp」、「写作资料」等 list，每个 list 都放了很多同类标签页。

每个 list 又可以贴不同的 label，如 Toby 的示例，我们可以将每个 list 贴上 「Work」或「Personal」的 label，从而让工作、生活泾渭分明。大家也可以按照自己需要，添加 「游戏」、「购物」等 label。

操作简单、直观

Toby 中管理标签页的方式有两种：

  1. 拖拽。你可以从右边列表中把已经打开的标签页拖入到左边的某个 list 中，对应标签页也会自动关闭。同样，标签页组织管理也可以用拖拽搞定。

  2. 一键保存。点击「Save Session」就可以将已经打开的所有标签页保存到一个 list 中。

搜索

随着标签页越来越多，搜索是比按照分类查看更快捷的查找方式。
Toby 不但可以搜索加入到 Toby 中的标签页，也可以搜索目前打开的所有标签页。

云同步

曾经，我常常在下班前面对一大堆想继续浏览的网页纠结要不要背电脑回家。

估计 Toby 开发者也有同样的遭遇，所以前些日子给 Toby 加入了同步功能。

再次推荐

如果你也是 Chrome 用户，如果你也被「两座大山」压着，如果你也希望有更好的标签页管理工具，那 Toby 绝对值得一试。
