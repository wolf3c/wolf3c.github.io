2020 目标

Posted on 2019-12-31，5 min read

我将自己的成长分为「内修」和「外建」两部分，2020 的计划，也将按照这两部分的成长展开。

「外建」部分的目标是：

  1. 让自己的产品能力更好，工作收入增长 5~10k；
  2. 尝试自己的 sideproject，收获 100 名忠实用户，并给自己能带来 1000 元/月的收入；
  3. 拥有 planB：与现在行业属性相对、经济属性相对、政治环境不同，应对世界突变；
  4. 购置保险，给自己为数不多的资产提供保障；
  5. 准备买房，为组建家庭做准备；

「内修」部分的目标：

  1. 提升专注力，让自己可以在工作场所，轻松持续专注 45 分钟；
  2. 养成良好的作息习惯，让自己更加精力旺盛、身体健康：
    1. 每天晚上 11 点入睡；
    2. 每天早上 7 点可以自然醒；
    3. 每周运动 5 次，每次 >30min；
    4. 每日冥想 ≥10 分钟；
  3. 每日读书 >半小时；
  4. 每日阅读一篇英文文章，提升自己的英文阅读能力；
  5. 了解并修习佛学、心学，用自己的话讲述自己了解的佛学、心学，各写一篇 blog；
  6. 学习经济学，至少读完《人的行为》，并写自己的读后感；

为了避免 2019 年计划中的「大跃进」问题，我们先看一下这些目标有没有不切实际的。

  1. 工作收入增长，靠努力也靠机遇，强求不得，但也不能因此坐等机遇，我应该着眼于自身成长，并不断寻找机会；
  2. planB 有很大的变数，自己现在没有完整计划，根据不同的项目，占用时间长短也大不相同，可能会出现没有时间做的情况；
  3. 买房，对个人财力压力很大，以我目前的实力，很难在明年去买房，截止目前，也没有想到可以准备的点；
  4. 11 点入睡，对于我来说并不容易，我希望在 3 月时，能养成这个习惯，至少达到每周有 5、6 天是 11 点入睡的；
  5. 《人的行为》是一本宏伟巨著，想要读完可真的不容易，不过，好在这是靠自己能个够克服的；
  6. 这么多事情，我的时间是否足够？

所以，

  1. 工作收入增长，我要将它变成一个成长计划；

  2. planB 我要做一个投入时间的计划，让自己有时间去做这个事情，至于做什么，根据当时的思考来做；

  3. 买房，我要做一个储蓄理财的规划，让自己财力上可以达到，另外，还要多去了解房市知识；

  4. 关于时间问题，可以计算一下：

    1. 所有必修内容的时间 = （30min 锻炼时间 + 30min 锻炼前后准备时间） + （10min 冥想 + 10min 前后准备和冗余）+ （ 30min 读书 + 20min 冗余） + （30min 阅读英文文章）= 2h 10min

    2. 关于工作占用时间：我自己 7 点起洗漱 1 小时，8 点上班，21 点半到 22 点到家，睡前 需要 30 分钟洗漱整理，所以，工作日没有事情做以上事情；

    3. 从这里来看，以上必修内容在非工作时间很难做，何况，planB、sideproject、了解生活其他知识、写 blog 等事情还没有计算时间，所以，现在能想到的办法就是1）砍掉这些事情；2）看看怎么在工作时间将这些事情完成；3）换个不加班的工作；4）换住处，减少通勤时间，但这顶多能挤出 1~2 小时；

    4. 关于周末，周六需要陪女朋友，周日需要加班（强度不及工作日）；

    5. 综上，我决定在工作时间拿出 3 小时给自己。因为自己的工作并不是那种高强度的搬砖工作，其实工作时间很长的一个原因是公司环境造成的，自己可能浪费了一些时间，所以，可以尝试挤一挤。具体做法：

      1. 早上读一篇英文文章（其他人看到也不会一下次知道内容）；
      2. 中午午休时间，冥想 10 分钟（环境安静，不会被打扰）；
      3. 下午18 点左右，去健身（吃饭时间，健身吃饭，不会被其他会议等留住）；
      4. 上下班路上，读书（我坐的一趟地铁很长，我也经常有座，反正我手机也没多少流量）；

      如果以上能达到，时间问题貌似能解决。

另外，需要避免笼统问题和缺少定期检查的问题，我需要做好两点：

  1. 将所有的事情做出时间计划和里程碑；
  2. 我要定期检查、复盘。（这里包括各种计划和每天或每周的周期性事项）

我需要做的计划包括：

  1. 读书计划：读哪些书、定量；
  2. 规律生活的调整计划：什么时候达到什么状态；
  3. sideproject 计划：什么时间完成什么事情，达成什么目标；
  4. 提升专注力的计划：如何提升，什么时候达到什么状态；
  5. planB 的计划：如何给出时间，如何尝试；
  6. 购置保险、准备买房等事项的计划；

关于定期复盘：

  1. 每周看周期性事项的完成情况（读书、英文阅读、健身、冥想），及时调整；
  2. 每月，看自己计划完成情况，分析完成或没有完成的原因，并做对应处理；

以上，作为一个 2020 计划的大体内容吧。
